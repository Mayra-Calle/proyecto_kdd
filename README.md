# Proyecto KDD con Streamlit

## Descripción
Proyecto de minería de datos basado en la metodología KDD utilizando Python y Streamlit.

## Etapas KDD implementadas
1. Selección
2. Preprocesamiento
3. Transformación
4. Minería de Datos
5. Evaluación
6. Presentación del Conocimiento

## Librerías utilizadas
- Streamlit
- Pandas
- Matplotlib
- Scikit-Learn

## Ejecución

```bash
streamlit run proyecto_kdd.py
```
